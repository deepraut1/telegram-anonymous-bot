# Telegram Anonymous Chat Bot

This bot deletes user messages and reposts them anonymously in the group.

## Features
- Deletes original message
- Reposts message with user's first name
- Supports text, photo, video

## Setup

1. Create bot using BotFather
2. Add bot to group as admin
3. Disable privacy mode (/setprivacy)

## Run Locally

```bash
pip install -r requirements.txt
python bot.py
